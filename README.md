# stealth-outbound

## Instructions to run:
- Clone this repository
- Install pipenv (<code>pip install pipenv</code>)
- <code>pipenv install</code>
- <code>pipenv shell</code>
- <code>streamlit run linkedin_app.py</code>
- App should appear automatically in your browser
